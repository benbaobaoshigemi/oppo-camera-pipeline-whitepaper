# OPPO Find X9 Ultra 相机色彩与画质管线技术白皮书

本仓库包含 OPPO Find X9 Ultra 移动成像管线的深度逆向工程分析白皮书。

## 文档目录

在 [whitepaper.md](whitepaper.md) 中，详细拆解并分析了以下模块：

1. **背景与架构概述**：整体渲染引擎的运作与拓扑结构。
2. **核心引擎：SimTool 与 LUT 系统**：逆向分析 `SimTool` 软色彩渲染器的配置与运行逻辑。
3. **普通拍照模式管线**：普通直出流程与硬件调校。
4. **大师模式 JPGmax 管线**：深度拆解大师模式的 10 大处理子层级（LMT, CWCM, SC, ODT, Vig, Merge, PostSR 等）。
5. **大师模式 RAW 与 RAWMax（计算 RAW）管线**：解析 TurboRAW 域融合、HybridRAW 神经网络超分、16-bit DNG 双光源色彩矩阵（ColorMatrix）以及 43,200 点 `HueSatMap` 标定细节。
6. **哈苏超清（50MP）管线**：高解像力商业摄影模式的 AI 降噪大模型细节。
7. **Gamma 曲线**：对比普通模式的激进提亮与大师模式的线性交权。
8. **多模式完整对比矩阵（含 RAWMax）**：全方位的多模式处理特征横向对比。
9. **HNCS 与大师模式的技术关系**：哈苏自然色彩在三维查找表与参数上的具体物理映射。
10. **完整管线流程图**：直观的拍照管线信号拓扑图。
